This is a nutrition website built with Java. It uses typescript and Angular for packaging.

## Technologies

- **Java Spring Boot**: An open-source tool that makes it easier to use Java-based frameworks to create microservices and web apps.
- **Angular**: A TypeScript-based free and open-source single-page web application framework used for packaging app to native.

This website is a nutrition website which provides personalized diet recipes for Authentic Indian Dishes.
