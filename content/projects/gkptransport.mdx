This is a portfolio website built Wordpress. It uses the `PHP wordpress`
package to render content in pages.

## Technologies

- **Wordpress**: A PHP framework for building static and dynamic websites.
- **GoDaddy Web Hosting**: A cloud platform for static sites and serverless functions.


This portfolio website is a great example of how you can use Wordpress to build a modern website with dynamic content. By combining the power of development you can create interactive and engaging websites that are easy to maintain and update.
